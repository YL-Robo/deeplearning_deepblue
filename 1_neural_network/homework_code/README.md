# 神经网络作业代码

## 文件说明

- `mnist_fcnn_solution.py` - MNIST手写数字识别完整解答（教学版）

## 运行方法

```bash
python mnist_fcnn_solution.py
```

## 依赖

```bash
pip install numpy matplotlib scikit-learn
```

## 网络结构

- 输入层: 784 (28×28)
- 隐藏层: 30
- 输出层: 10
- 激活函数: Sigmoid

## 预期结果

- 训练准确率: ~95%
- 测试准确率: ~92-94%
- 训练时间: 约2-3分钟
